# Product Overview

Personal portfolio website for Immanuel Wandeto, a web designer/developer based in Kenya.

## Purpose
Showcase web development and UI design projects to potential clients, demonstrating capabilities in:
- Landing pages and business websites
- E-commerce solutions with payment integration
- Web applications with admin dashboards
- Mobile UI/UX design (Figma)

## Target Audience
Business owners seeking professional web presence, from restaurants and nightclubs to driving schools and service companies.

## Key Features
- Single-page portfolio layout
- Project showcase with live demo links
- Contact information and social links
- Responsive design with parallax effects
